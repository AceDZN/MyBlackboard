# MyBlackboard Widget
Just another dashboard :)

####Familiar with Git?#####
Checkout this repo, install dependencies, then start the http-server process with the following:

```
	> git clone git@github.com:AceDZN/comments_widget.git
	> cd comments_widget
	> npm install
	> gulp
```
